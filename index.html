Abstract

Stylized voxel art is widely used in games and digital media, but generating visually appealing voxel representations from 3D meshes remains challenging and often requires manual effort. Existing methods struggle to preserve semantic structure and offer limited control over stylization. We propose Voxify3D, a differentiable two-stage framework for generating stylized voxel art from 3D meshes. The first stage initializes a coarse voxel grid via neural volume rendering, while the second stage refines it using six-view orthographic pixel art supervision and a discrete color palette from clustering methods (e.g., K-means, Max-Min, Median Cut). We introduce Gumbel-Softmax-based differentiable palette quantization and a CLIP-based perceptual loss to ensure semantic alignment. Voxify3D allows flexible control over voxel resolution, color selection, and palette size, producing expressive outputs with strong structural and stylistic consistency. Experiments and user studies demonstrate that Voxify3D outperforms prior methods in visual quality, semantic fidelity, and pixel-level aesthetics.

Pipeline

Our two-stage voxel art generation pipeline. (a) Coarse voxel grid training: Given a 3D mesh, we render multi-view images and optimize a voxel-based radiance field (DVGO [Sun et al. 2022]) using MSE loss to learn coarse geometry and appearance. (b) Orthographic pixel art fine-tuning: We refine the voxel grid using six orthographic pixel art views, which also serve to extract a discrete color palette (e.g., via k-means). Optimization includes appearance, depth, and alpha losses. (c) CLIP-guided optimization: A CLIP loss computed over rendered patches and mesh images encourages semantic alignment while being memory-efficient. (d) Differentiable discrete color selection via Gumbel-Softmax: Each voxel stores palette logits. Gumbel-Softmax enables differentiable sampling for end-to-end color optimization, yielding coherent, stylized voxel art.
